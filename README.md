# GLE ⚡

GLE is a modern and responsive competition platform built with React.js and Tailwind CSS. The platform provides users with an engaging interface to explore competitions, events, and opportunities with a smooth and user-friendly experience.

🌐 **Live Demo:**  
https://gle-z.netlify.app/

---

- 🎯 Modern UI/UX Design
- 📱 Fully Responsive Layout
- ⚡ Fast and Smooth Performance

- 🚀 Optimized Frontend Performance

# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/rajibulislam60/gle.git
```

Go to the project directory:

```bash
cd gle
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
